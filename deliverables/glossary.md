# Glossary of Acronyms & Terms

Every acronym and specialist term used across the nine papers, alphabetical. Where a term maps to more than one thing, both meanings are listed and disambiguated.

## Philippine government

| Term | Expansion | Notes |
|---|---|---|
| **ABC** | Approved Budget for the Contract | The procurement ceiling. For this bid: PHP 500 M. See [Paper A §A.1](A_technical_specifications_brief.md#a1-project-fundamentals). |
| **ARMM / BARMM** | Autonomous Region in Muslim Mindanao / Bangsamoro Autonomous Region | Special-status region with distinct HR rules. [Paper H §H.6](H_data_migration.md#h6-wave-plan--regional-rollout-not-big-bang). |
| **BAC** | Bids and Awards Committee | The DepEd body running the procurement. |
| **BIR** | Bureau of Internal Revenue | Tax authority. Regulator #5 in [Paper K](K_integration_specifications.md#k42-per-regulator-specifications). |
| **CO** | Central Office | DepEd HQ in Makati; a top-level organisational unit. |
| **COA** | Commission on Audit | The state audit body. |
| **COS** | Contract of Service | Non-plantilla employment category. |
| **CSC** | Civil Service Commission | Regulator #6; owns CSC Form 9, PDS, eligibility rules. |
| **CSC Form 9** | Position posting form | Required for every plantilla posting. See [Paper A §A.4 Module 2](A_technical_specifications_brief.md#module-2--job-posting-tor-52). |
| **DBM** | Department of Budget and Management | Owns PSIPOP. Regulator #7. |
| **DCP** | DepEd Computerization Program | Funding stream (via SAGIP). |
| **DepEd** | Department of Education | The PIC and contracting entity. |
| **DICT** | Department of Information and Communications Technology | Owns GovCloud PH; regulator #8 in the compliance sense. |
| **DPO** | Data Protection Officer | Required by RA 10173 §19. See [Paper I](I_privacy_impact_assessment.md). |
| **eGovPH** | eGovernment PH portal | DICT-run SSO / interop portal. |
| **eKYC** | Electronic Know-Your-Customer | Digital identity verification via PhilSys. |
| **GAA** | General Appropriations Act | Annual national budget. Funding source for the DCP–SAGIP line. |
| **GSIS** | Government Service Insurance System | Retirement / loan authority. Regulator #1. |
| **GovCloud PH** | Philippine government cloud | DICT-managed sovereign cloud. Referenced in [Paper F §F.5.3](F_delivery_and_cost.md#f53-dr-site--cost-reduced-hot-standby). |
| **HDMF** | Home Development Mutual Fund | Formal name of Pag-IBIG. |
| **HR** | Human Resources | Function; also a role type (HR Officer). |

| **HRO** | Human Resource Officer | HR staff at CO, RO, SDO, District, or School level. |
| **ICTS-SDD** | Information and Communication Technology Service — Solutions Development Division | The DepEd unit procuring the HRIS. |
| **IPCR** | Individual Performance Commitment and Review | Component of SPMS. |
| **JHS** | Junior High School | School-level classification. |
| **JO** | Job Order | Non-plantilla employment category. |
| **LIS** | Learner Information System | DepEd system; **not** the HRIS. |
| **MoA** | Memorandum of Agreement | Legal instrument between agencies. |
| **NPC** | National Privacy Commission | RA 10173 regulator. Receives PIA filings. |
| **NTP** | Notice to Proceed | Contract-start signal. The 365 days count from NTP. |
| **OPCR** | Office Performance Commitment and Review | Aggregate version of IPCR. |
| **Pag-IBIG** | HDMF branded name | Regulator #2. |
| **PBD** | Philippine Bidding Document | The source RFP. `PBD_HRIS_For-Posting.pdf` in this repo. |
| **PDS** | Personal Data Sheet | CSC standard employee form. |
| **PhilGov cert** | Philippine Government-issued certificate | Code-signing / mTLS. |
| **PhilHealth** | Philippine Health Insurance Corporation | Regulator #3. |
| **PhilSys** | Philippine Identification System | National ID system, PSA-run. Regulator #9. |
| **PIC** | Personal Information Controller | RA 10173 role — DepEd is the PIC. |
| **PIP** | Personal Information Processor | RA 10173 role — the prime contractor. |
| **PIS** | Personnel Information System | The legacy DepEd system being replaced. |
| **PIS (RA 10173)** | Personal Information System | Same acronym; means a system that processes personal data. Distinguished from context. |
| **PMP** | Project Management Professional | Certification. |
| **PMR** | Project Monitoring Report | Standard procurement reporting artefact. |
| **PSA** | Philippine Statistics Authority | Runs PhilSys. |
| **PSIPOP** | Position Structure and Incumbency of Personnel of Positions | DBM's plantilla master file. |
| **RA 9184** | Government Procurement Reform Act | The umbrella procurement law. |
| **RA 9470** | National Archives Act | Records-retention regulator. |
| **RA 10173** | Data Privacy Act of 2012 | The privacy law. See [Paper I](I_privacy_impact_assessment.md). |
| **RO** | Regional Office | Second-level DepEd organisational unit. |
| **RSP** | Recruitment, Selection, and Placement | Module 3 of the HRIS. |
| **SAGIP** | Sustained Assistance and Guidance for Improved Performance | The DCP continuing fund line funding this bid. |
| **SCC** | Special Conditions of Contract | PBD section; e.g. SCC 7 forbids subcontracting. |
| **SDO** | Schools Division Office | Third-level DepEd organisational unit. |
| **SHS** | Senior High School | School-level classification. |
| **SPMS** | Strategic Performance Management System | CSC-mandated performance framework. |
| **SSS** | Social Security System | Regulator #4; covers COS / JO staff. |
| **TIN** | Tax Identification Number | BIR-issued. |
| **TOR** | Terms of Reference | Section VII of the PBD. |
| **UAT** | User Acceptance Testing | M7 milestone activity. |
| **WES** | Work Experience Sheet | CSC applicant document. |
| **WFP** | Workforce Profile | DBM plantilla / assignment view. |

## Financial / rate abbreviations

| Term | Notes |
|---|---|
| **PHP** | Philippine peso. All costs in these papers unless stated. |
| **USD** | US dollar. Applies where foreign software, cloud, or hardware is priced in dollars. |
| **13th month** | Statutory year-end bonus in the Philippines. Included in the 1.35 loading factor. |
| **Loaded rate** | Base salary plus statutory contributions and benefits. See [Paper F §F.3.1](F_delivery_and_cost.md#f31-reference-rates--2026-ph-market-fully-loaded). |
| **T&M** | Time and Materials contracting model. |
